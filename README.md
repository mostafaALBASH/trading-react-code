## Getting Started


```shell
git clone git@github.com:mostafaALBASH/tradeling-coding-react.git 
cd 
npm install
npm start

$ git clone https://github.com/mostafaALBASH/tradeling-coding-react.git
$ cd tradeling-coding-react
$ npm install
$ npm start
```
